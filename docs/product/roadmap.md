# Roadmap

> Hacia dónde va dev-setup-mac-es.
> **Última actualización**: 2026-07-03

Este proyecto crece agregando scripts para más herramientas del entorno de
desarrollo y puliendo los existentes. No hay fechas comprometidas; el orden es
orientativo.

## Hecho (v1.0.0)

- 13 scripts: base, zsh, git, ssh, node, python, ruby, java, postgresql, docker,
  terraform, kubernetes.
- Estructura de documentación y gobernanza.
- CI con ShellCheck.

## Próximo

- [ ] Nuevos lenguajes/herramientas frecuentes (p. ej. Go, Rust).
- [ ] Verificación de integridad (checksums/firmas) en las descargas directas.
- [ ] Opción de ejecución no interactiva (variables de entorno en vez de prompts).

## Ideas / a evaluar

- [ ] Script orquestador opcional para instalación completa desatendida.
- [ ] Soporte y pruebas en más versiones de macOS (Intel y Apple Silicon).
- [ ] Matriz de CI probando la ejecución real en runners de macOS.
