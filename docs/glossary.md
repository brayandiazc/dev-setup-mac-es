# Glosario

Vocabulario compartido del proyecto. Definiciones cortas para que todos usen los
términos de la misma forma. Ordenado alfabéticamente.

| Término           | Definición                                                                                  |
| ----------------- | ------------------------------------------------------------------------------------------- |
| Apple Silicon     | Chips ARM de Apple (M1, M2, M3…). Homebrew se instala en `/opt/homebrew`.                    |
| Cask              | Fórmula de Homebrew para instalar apps de macOS con GUI (`brew install --cask`).            |
| CLT               | Xcode Command Line Tools: compiladores y utilidades base que muchos scripts necesitan.      |
| Homebrew          | Gestor de paquetes de macOS. Base de casi todos los scripts.                                |
| Idempotencia      | Propiedad de un script que puede ejecutarse varias veces con el mismo resultado seguro.     |
| Intel             | Macs con procesador Intel. Homebrew se instala en `/usr/local`.                             |
| minikube          | Herramienta que levanta un clúster de Kubernetes local para desarrollo.                     |
| nodenv            | Gestor de versiones de Node.js.                                                             |
| Oh My Zsh         | Framework de configuración y plugins para Zsh.                                              |
| Postgres.app      | Distribución de PostgreSQL como app de macOS, con arranque/parada por GUI.                  |
| pyenv             | Gestor de versiones de Python.                                                              |
| rbenv             | Gestor de versiones de Ruby.                                                                |
| SDKMAN!           | Gestor del ecosistema Java (JDK, Maven, Gradle…).                                           |
| Zsh               | Shell por defecto de macOS moderno; el que configura este proyecto.                         |

> Convención: enlaza al documento donde se detalla cada concepto cuando aplique.
