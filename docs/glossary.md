# Glosario

Vocabulario del proyecto y de las herramientas que instala. Mantén las
definiciones cortas y sin ambigüedad.

| Término       | Definición                                                                                     |
| ------------- | ---------------------------------------------------------------------------------------------- |
| Apple Silicon | Chips ARM de Apple (M1/M2/M3…); Homebrew se instala en `/opt/homebrew`.                          |
| Cask          | Fórmula de Homebrew para instalar apps de macOS con GUI (`brew install --cask`).               |
| Docker        | Plataforma de contenedores para empaquetar y ejecutar aplicaciones aisladas.                   |
| Homebrew      | Gestor de paquetes de macOS; base de casi todos los scripts.                                    |
| Idempotencia  | Propiedad de un script que puede ejecutarse varias veces produciendo el mismo estado final.    |
| Intel         | Macs con procesador Intel; Homebrew se instala en `/usr/local`.                                 |
| minikube      | Herramienta que levanta un clúster de Kubernetes local para desarrollo.                         |
| nodenv        | Gestor de versiones de Node.js (permite múltiples versiones por proyecto).                      |
| Oh My Zsh     | Framework de configuración y plugins para el shell Zsh.                                         |
| Postgres.app  | Distribución de PostgreSQL como app de macOS, con arranque/parada por GUI.                      |
| pyenv         | Gestor de versiones de Python.                                                                  |
| rbenv         | Gestor de versiones de Ruby.                                                                    |
| SDKMAN!       | Gestor de SDKs de la JVM (Java, Maven, Gradle, etc.) usado para instalar Java.                  |
| SSH           | Protocolo para autenticación y conexión remota segura (aquí, con GitHub).                       |
| Zsh           | Shell interactivo avanzado, alternativa a Bash.                                                 |

> Convención: ordena los términos alfabéticamente.
